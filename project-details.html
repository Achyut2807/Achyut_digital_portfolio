Technical Project Details

Detailed engineering implementation and technical workflow of selected projects

01. FPGA-Based Data Acquisition System

Oct 2025 – Present

This project focuses on developing a deterministic FPGA-based data acquisition framework designed for reliable multi-clock digital signal capture under high-speed operating conditions.

  • Designed Verilog HDL based acquisition architecture using FIFO buffering.
  • Implemented clock-domain crossing for asynchronous data transfer.
  • Applied Vivado XDC constraints for setup and hold timing verification.
  • Validated deterministic data capture through timing closure analysis.

02. Wavelet-Based Vital Signs Detection

Jul 2025 – Nov 2025

A MATLAB-based non-contact RF sensing framework developed for extracting respiration and heartbeat signals under cluttered environments for disaster rescue applications.

  • Applied multirate wavelet decomposition for signal separation.
  • Separated respiration band (0.2–0.5 Hz) and heartbeat band (1–2 Hz).
  • Handled low SNCR and clutter effects during detection.
  • Integrated UAV and Intelligent Reflecting Surface assisted sensing concept.

03. Smart Vehicular Safety System

Nov 2025 – Present

This project develops an intelligent vehicular safety framework using embedded sensing and machine learning for rider behavior monitoring and predictive safety alerts.

  • Integrated ESP32 with BNO055 IMU and GNSS/GSM modules.
  • Captured acceleration, braking and speed profiles in real time.
  • Implemented DNN-MLP based detection for rash driving and harsh braking.
  • Used MQTT telemetry for cloud-based logging and long-term rider analysis.

04. Wi-Fi Campus Network Design

Feb 2025 – Jun 2025

Designed a large-scale smart campus wireless network architecture supporting high-density user environments and multiple simultaneous devices.

  • Designed Wi-Fi 5/6 network for 1000+ students and 4000+ devices.
  • Analyzed throughput, latency, fairness and utilization using MATLAB.
  • Applied OFDM and MU-MIMO based spatial multiplexing concepts.
  • Implemented VLAN segmentation and secure wireless access design.
